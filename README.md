# FingerDraw

FingerDraw is an application that turns your Android device into a drawing tablet for your PC.

## Components

*   **Android Client:** The Android application that captures touch input.
*   **Linux Server:** The server application for Linux that receives input and mirrors the screen.
*   **Windows Server:** The server application for Windows that receives input and mirrors the screen.

## How to Use

1.  Start the server on your PC (either the Linux or Windows version).
2.  The server will display its IP address.
3.  Enter this IP address into the Android app and tap "Connect".
4.  The screen of your PC will be mirrored on your Android device, and you can control the mouse with your finger or a stylus.
